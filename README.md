# Lyrics





## Usage
Install [nodemon](https://github.com/remy/nodemon) globally

```
npm i nodemon -g
```

Install server and client dependencies

```
npm install
cd client
npm install
```

To start the server and client at the same time (from the root of the project)

```
npm run dev
```
